Jede Programmiersprache besitzt eine sogenannte API (Application Programming Interface). Eine API ist ein Satz von Befehlen, Funktionen, Protokollen und Objekten, die Programmierer verwenden können, um eine Software zu erstellen oder mit einem externen System zu interagieren. Sie stellt Entwicklern Standardbefehle für die Ausführung allgemeiner Operationen zur Verfügung, so dass Codes nicht von Grund auf neu geschrieben werden müssen.

Je weniger Standardbefehle eine Sprache hat, desto einfacher ist sie zu erlernen. Aber auch desto schwieriger ist es, komplexe Befehle herzustellen, da man sie selbst bauen muss.

APIs werden mit den Versionen der Sprache angepasst. Idealerweise hat dies keinen Einfluss auf Programme, die mit einer älteren Version der Sprache geschrieben wurden, jedoch ist dies nicht immer so. Was durchaus zu Problemen führt.

Die API von PyTamaro beschränkt sich auf die primitiven Grafiken (atomare Grafikbausteine, aus denen man komplexe Bilder zusammensetzen kann) und einige vorgegebene Funktionen und Konstanten. Im Unterricht brauchen wir die grafische, abgespeckte Version der API von PyTamaro, die zum Erlernen der Sprache einfacher ist.
