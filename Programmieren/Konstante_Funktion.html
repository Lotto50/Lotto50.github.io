Unterschied Konstante und Funktion

Konstante

Eine Konstante ist ein Code Abschnitt, welcher nicht verÄndert werden kann. Er ist also nicht variabel. Der Programm Code wird fix ausgegeben und wird nicht durch Eingaben verändert.

Funktion

Eine Funktion ist variabel und somit verÄndert werden. Der Code kann leicht angepasst werden und wird dadurch jedesmal neu ausgegeben.

Die Funktion wird oft gebraucht, wenn man einfach und schnell die Farbe von Objekten ändern will.
