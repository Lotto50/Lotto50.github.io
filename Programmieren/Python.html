Python ist eine vielseitig einsetzbare höhere Programmiersprache und gehört zu den am weitesten verbreiteten Programmiersprachen (untenstehende Grafik zeigt den PYPL-Index (Popularity of Programming Language Index) vom September 2023, welcher die Beliebtheit einer Programmiersprache an den Online-Suchanfragen bemisst und wonach Python die klare Nr. 1 unter den Programmiersprachen ist; andere Indices verwenden andere Kriterien für die Beliebtheit und stufen Java oder JavaSkript (grau) als die am weitesten verbreitete Sprache ein.). Sie zeichnet sich durch eine einfache und übersichtliche Syntax aus und ist zu verschiedenen Programmierstilen kompatibel. Python wurde 1994 von Guido van Rossum entwickelt, heute wird die Weiterentwicklung durch die Python Software Foundation koordiniert und unter einer OpenSource-Lizenz veröffentlicht. Benannt wurde Python nach der englischen Komikertruppe Monty Python.
