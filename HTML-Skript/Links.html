Um einen Link auf eine andere Seite einzubinden, gibt es das Tag <a>, als Attribut href wird diesem die Adresse der zu verlinkenden Website (als URL oder als Pfad zu einem Dokument auf dem gleichen Server wie die aktuelle Seite) angegeben. Um einen Link auf die Webseite unserer Schule zu erstellen, müsste dieser Code eingegeben werden:

<a href="https://www.ksalpenquai.lu.ch"> meine Schule </a>

was dann folgende Ausgabe ergibt: meine Schule.

Ohne spezielle Erwähnung (d.h. by default) werden die verlinkten Seiten im selben Browser-Fenster geöffnet. Damit die verlinkte Seite in einem neuen Fenster geöffnet wird, wird dem Link-Tag ein Attribut hinzugefügt:

<a href="../Skript.html" target="_blank"> Skript-Startseite </a>

was dann folgende Ausgabe ergibt: Skript Startseite
