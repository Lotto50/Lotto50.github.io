Bilder können und müssen nicht in HTML-Code beschrieben werden (wobei es Ausnahmen gibt, zum Beispiel ergibt &#129409; dieses Bild: 🦁), sondern werden über Bilddateien eingebunden. Ein Bildelement wird mit einem Bildtag angegeben, welches den Pfad zur Bilddatei angibt:

<img src="./Bild.png" >

Mit diesem Tag wird ein Bild eingefügt, welches unter dem Namen Bild.png abgespeichert ist. Das img-Tag kann verschiedene Attribute haben. Der folgende Code fügt das unter Bild.png gespeicherte Bild mit einer zur Seitenbreite relativen Breite von 50% und einer Höhe von 400 ein. Wenn entweder nur die Höhe oder die Breite bestimmt wird, werden die Längenverhältnisse des ursprünglichen Bildes erhalten. Das Attribut alt gibt einen Text an, welcher an der Stelle des Bildes erscheint, wenn auf dieses aus irgendeinem Grund nicht zugegriffen werden kann.

<img src="./Bild.png" width="50%" "height=400" alt="ein Bild">
