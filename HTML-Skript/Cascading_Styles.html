CSS bdeutet Cascading Style Sheets. HTML-Dokumente haben meistens eine verschachtelte Struktur. Für jedes der ineinandergeschachtelten Elemente können im Stildokument Regeln beschrieben werden. Dabei gilt, dass eine Regel, welche für ein Ober-Element beschrieben wird, auch für alle seine Unter-Elemente gilt. Die definierten Eigenschaften werden vom Ober-Element an die Unter-Elemente vererbt. Wenn aber eine für ein Unter-Element eine andere Regel definiert wird, so gilt für dieses Element diese Regel. Die für ein spezifisches Unter-Element definierte Regel überstimmt die Regel des Ober-Elements. Die Regeln für die verschiedenen Elemente bilden eine Kaskade.
